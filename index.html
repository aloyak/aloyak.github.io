Origin Engine

C++20 OpenGL SDL2 ImGui CMake

A cross-platform true-3D-pixelart game engine built in C++20 with OpenGL and SDL2. Includes a custom sandbox editor built with ImGui for creating and editing scenes visually.

Features dynamic lighting, an entity-component system with JSON scene serialization, basic physics and collisions (Bullet3), an Audio System (SoLoud) and a clean developer-friendly API.

This engine combines many different technologies and concepts I've been interested in, and it's been a great learning experience building it from scratch.
